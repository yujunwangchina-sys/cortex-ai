# 文档预览组件集成指南

本指南介绍如何在 CortexAI 平台中集成文档预览功能，支持 PDF、Word、Excel、PowerPoint 和 HTML 文件预览。

---

## 📦 第一步：安装依赖

```bash
cd Cortex-Vue3

# 安装 Word 和 Excel 预览库
npm install docx-preview xlsx
```

---

## 📥 第二步：安装 PDF.js

### 方式 1：手动下载（推荐）

1. 访问 [PDF.js Releases](https://github.com/mozilla/pdf.js/releases)
2. 下载最新版本（如 `pdfjs-3.11.174-dist.zip`）
3. 解压后，将内容复制到 `Cortex-Vue3/public/pdfjs/` 目录

目录结构应该是：
```
public/
└── pdfjs/
    ├── build/
    │   ├── pdf.js
    │   ├── pdf.worker.js
    │   └── ...
    └── web/
        ├── viewer.html
        ├── viewer.css
        └── ...
```

### 方式 2：使用脚本下载

**Linux/Mac:**
```bash
cd Cortex-Vue3/public
mkdir -p pdfjs
cd pdfjs
wget https://github.com/mozilla/pdf.js/releases/download/v3.11.174/pdfjs-3.11.174-dist.zip
unzip pdfjs-3.11.174-dist.zip
rm pdfjs-3.11.174-dist.zip
```

**Windows (PowerShell):**
```powershell
cd Cortex-Vue3\public
New-Item -ItemType Directory -Force -Path pdfjs
cd pdfjs
Invoke-WebRequest -Uri "https://github.com/mozilla/pdf.js/releases/download/v3.11.174/pdfjs-3.11.174-dist.zip" -OutFile "pdfjs.zip"
Expand-Archive -Path pdfjs.zip -DestinationPath .
Remove-Item pdfjs.zip
```

## 🚀 第三步：使用组件

### 在知识库或文件管理中集成

假设你要在知识库的文件预览中使用：

```vue
<template>
  <div class="knowledge-file-list">
    <!-- 文件列表 -->
    <el-table :data="files">
      <el-table-column prop="fileName" label="文件名" />
      <el-table-column label="操作">
        <template #default="{ row }">
          <el-button @click="previewFile(row)">预览</el-button>
          <el-button @click="downloadFile(row)">下载</el-button>
        </template>
      </el-table-column>
    </el-table>

    <!-- 预览对话框 -->
    <el-dialog
      v-model="previewVisible"
      :title="`预览: ${currentFile.fileName}`"
      width="90%"
      :destroy-on-close="true"
    >
      <div style="height: 70vh;">
        <DocViewer
          :fileUrl="currentFile.fileUrl"
          :fileName="currentFile.fileName"
        />
      </div>
    </el-dialog>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import DocViewer from '@/components/DocViewer'
import request from '@/utils/request'

const files = ref([])
const previewVisible = ref(false)
const currentFile = ref({})

// 加载文件列表
const loadFiles = async () => {
  const res = await request({ url: '/knowledge/files' })
  files.value = res.data
}

// 预览文件
const previewFile = (file) => {
  currentFile.value = {
    fileName: file.fileName,
    fileUrl: `/profile/knowledge/${file.filePath}`  // 根据实际路径调整
  }
  previewVisible.value = true
}

// 下载文件
const downloadFile = (file) => {
  window.open(file.fileUrl, '_blank')
}

loadFiles()
</script>
```

## 🔧 第四步：后端配置

确保文件路径可以访问：

### 1. Spring Boot 静态资源配置

检查 `application.yml`：

```yaml
cortex:
  # 文件上传路径
  profile: D:/cortex/uploadPath  # Windows
  # profile: /home/cortex/uploadPath  # Linux
```

### 2. 静态资源映射

检查 `ResourcesConfig.java`：

```java
@Override
public void addResourceHandlers(ResourceHandlerRegistry registry)
{
    /** 本地文件上传路径 */
    registry.addResourceHandler(Constants.RESOURCE_PREFIX + "/**")
            .addResourceLocations("file:" + CortexConfig.getProfile() + "/");
}
```

### 3. 跨域配置（如果需要）

如果文件和前端不在同一域名，需要配置 CORS：

```java
@Bean
public CorsFilter corsFilter()
{
    CorsConfiguration config = new CorsConfiguration();
    config.addAllowedOriginPattern("*");
    config.addAllowedHeader("*");
    config.addAllowedMethod("*");
    config.setMaxAge(1800L);
    
    UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
    source.registerCorsConfiguration("/**", config);
    return new CorsFilter(source);
}
```

## 📋 支持的文件格式

| 格式 | 扩展名 | 预览效果 | 备注 |
|------|--------|----------|------|
| **PDF** | `.pdf` | ⭐⭐⭐⭐⭐ | 完美支持，可缩放、搜索 |
| **Word** | `.docx` | ⭐⭐⭐⭐ | 样式还原度高，`.doc` 需转换 |

| **文本** | `.txt`, `.md` | ⭐⭐⭐⭐⭐ | 可扩展支持 |

## ⚙️ 高级配置

### 1. 自定义 Word 渲染

编辑 `DocViewer/index.vue`，修改 `loadDocx` 方法：

```javascript
await renderAsync(blob, docxContainer.value, null, {
  className: 'docx-wrapper',
  inWrapper: true,
  breakPages: true,        // 是否分页
  ignoreWidth: false,      // 是否忽略宽度
  ignoreHeight: false,     // 是否忽略高度
  ignoreFonts: false,      // 是否忽略字体
  trimXmlDeclaration: true
})
```

### 2. Excel 样式定制

修改 Excel 表格样式：

```css
.sheet-content :deep(table) {
  font-size: 12px;
}

.sheet-content :deep(td) {
  padding: 6px 8px;
  white-space: nowrap;
}

.sheet-content :deep(th) {
  background: #409eff;
  color: #fff;
}
```

### 3. 添加图片预览

扩展 `DocViewer` 组件支持图片：

```vue
<!-- 在 template 中添加 -->
<div v-else-if="['jpg', 'jpeg', 'png', 'gif', 'webp'].includes(fileType)" class="image-viewer">
  <img :src="fileUrl" alt="预览图片" />
</div>

<!-- 添加样式 -->
<style>
.image-viewer {
  display: flex;
  align-items: center;
  justify-content: center;
  height: 100%;
  background: #000;
}

.image-viewer img {
  max-width: 100%;
  max-height: 100%;
  object-fit: contain;
}
</style>
```

## 🐛 常见问题

### Q1: PDF 显示空白

**原因**：PDF.js 文件路径不正确

**解决**：
1. 检查 `public/pdfjs/web/viewer.html` 是否存在
2. 查看浏览器控制台是否有 404 错误
3. 确认 `pdfUrl` 计算属性中的路径正确

### Q2: Word 文档样式不对

**原因**：字体缺失或复杂样式不支持

**解决**：
- 在服务器安装常用中文字体
- 调整 `renderAsync` 选项
- 复杂文档建议使用 Office Online Viewer

### Q3: PowerPoint 无法预览

**原因**：使用微软在线服务需要文件公网可访问

**解决**：
- 方案 1：将文件部署到公网可访问的位置
- 方案 2：使用 [PPTXjs](https://github.com/meshesha/PPTXjs) 等客户端库
- 方案 3：转换为 PDF 后预览

### Q4: Excel 列太多显示不全

**解决**：添加横向滚动

```css
.sheet-content {
  overflow-x: auto;
  max-width: 100%;
}
```
