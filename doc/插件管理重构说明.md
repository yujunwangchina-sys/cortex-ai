## 使用说明

### 加载MCP包

1. 点击工具栏"加载MCP包"按钮
2. 系统自动扫描本地Python和Node.js环境中已安装的MCP包
3. 选择要加载的包，点击"加载"按钮
4. 包会自动添加到插件列表中
5. 在插件列表中点击"修改"进行详细配置（环境变量、审批等）

### 下载MCP包

1. 点击工具栏"下载MCP包"按钮
2. 选择运行时类型（uvx或npx）
3. 输入包名，如：
   - Python: `mcp-server-sqlite`
   - Node.js: `@modelcontextprotocol/server-filesystem`
4. 可选填写版本号，留空则安装最新版本
5. 点击"开始下载"
6. 下载完成后自动加载到插件列表

### 配置插件

在插件列表中：
1. 选择插件，点击"修改"按钮
2. 配置环境变量（JSON格式）
3. 设置是否需要审批
4. 配置其他信息（描述、备注等）
5. 保存

## API接口

### 扫描MCP包
```
GET /plugin/list/scanMcpPackages
```

### 下载Python包
```
POST /plugin/package/python/install
{
  "packageName": "mcp-server-sqlite",
  "version": "1.0.0",
  "useMirror": true
}
```

### 下载Node.js包
```
POST /plugin/package/node/install
{
  "packageName": "@modelcontextprotocol/server-filesystem",
  "version": "1.0.0",
  "useMirror": true
}
```

### 加载MCP包到插件列表
```
POST /plugin/list/enableMcpPackage
{
  "packageName": "mcp-server-sqlite",
  "runtimeType": "uvx",
  "envVars": null,
  "requireApproval": "0"
}
```

## 注意事项

1. **环境要求**：
   - Python环境：需要安装Python和pip
   - Node.js环境：需要安装Node.js和npm

2. **网络要求**：
   - 下载MCP包需要网络连接
   - 建议使用镜像源加速下载

3. **权限要求**：
   - 需要 `plugin:list:add` 权限才能加载和下载MCP包
   - 需要 `plugin:list:edit` 权限才能修改插件配置
