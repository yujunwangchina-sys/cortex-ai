## 🔧 具体改动

### 1. AgentChat.vue

#### 新增功能
```javascript
// 移动端检测
const isMobile = ref(false)
const checkMobile = () => {
  isMobile.value = window.innerWidth <= 768
  if (isMobile.value) {
    sidebarCollapsed.value = true
  }
}

onMounted(() => {
  checkMobile()
  window.addEventListener('resize', checkMobile)
})
```

## 🎨 关键CSS技巧

### 1. 侧边栏抽屉效果
```css
/* 使用transform性能更好 */
.sidebar-wrapper {
  transform: translateX(0);
  transition: transform 0.3s ease;
}
.sidebar-wrapper.collapsed {
  transform: translateX(-100%);
}
```

### 2. 响应式隐藏/显示
```css
/* 桌面端隐藏 */
.close-btn {
  display: none;
}

/* 移动端显示 */
@media (max-width: 768px) {
  .close-btn {
    display: flex;
  }
}
```

### 3. 文字选择性隐藏
```css
/* 桌面端: 图标+文字 */
.toolbar-btn {
  display: flex;
  gap: 4px;
}

/* 移动端: 仅图标 */
@media (max-width: 768px) {
  .toolbar-btn .btn-text {
    display: none;
  }
}
```

## 🚀 性能优化点

1. ✅ **使用transform代替left/right** - 硬件加速
2. ✅ **transition仅在必要属性上** - 减少重绘
3. ✅ **移动端减少阴影效果** - 提升渲染性能
4. ✅ **合理使用fixed定位** - 避免频繁重排
5. ✅ **图片懒加载和压缩** - 减少流量消耗

// AgentChat.vue
const isMobile = ref(false)           // 是否移动端
const sidebarCollapsed = ref(false)   // 侧边栏是否折叠

// 初始化逻辑
onMounted(() => {
  checkMobile()
  window.addEventListener('resize', checkMobile)
})

function checkMobile() {
  isMobile.value = window.innerWidth <= 768
  if (isMobile.value) {
    sidebarCollapsed.value = true  // 移动端默认折叠
  }
}
