| ≤480px | 小屏手机 | 更紧凑的布局 + 更小字体 |

## 🎓 技术亮点

### 1. CSS媒体查询
使用标准的CSS3媒体查询实现响应式:
```css
@media (max-width: 768px) { /* 移动端样式 */ }
@media (max-width: 480px) { /* 小屏样式 */ }
```

### 2. Flexbox布局
充分利用Flexbox实现灵活布局:
```css
.agent-chat { display: flex; }
.chat-main { flex: 1; min-width: 0; }
```

### 3. CSS Transform
使用transform实现硬件加速动画:
```css
.sidebar-wrapper.collapsed { transform: translateX(-100%); }
```

### 4. Shadow DOM
Widget使用Shadow DOM实现样式隔离:
```javascript
shadow = host.attachShadow({ mode: 'open' });
```

### 5. 触摸优化
```css
/* iOS惯性滚动 */
-webkit-overflow-scrolling: touch;

/* 触摸反馈 */
:active { transform: scale(0.95); }
```
