# AgentChat 界面优化说明

## 设计理念

参考 OpenUI 的设计风格，采用**海尔蓝**作为主色调，打造简洁、现代、以对话为中心的聊天界面。

## 主色调：海尔蓝

```css
--haier-blue: #006BB7;          /* 主色 */
--haier-blue-light: #1E88E5;    /* 亮色 */
--haier-blue-lighter: #E3F2FD;  /* 浅色背景 */
--haier-blue-dark: #004C8C;     /* 深色 */
```

## 核心优化内容

### 1. 整体布局
- **三栏式布局**：侧边栏（会话列表） + 主对话区 + 无独立右侧栏
- **高度适配**：75vh，支持响应式（桌面/平板/手机）
- **圆角设计**：16px 圆角，提升现代感
- **渐变背景**：`linear-gradient(to bottom, #f5f9fc, #e8f1f8)`

### 2. 侧边栏优化
- **顶部海尔蓝渐变**：`linear-gradient(135deg, #006BB7, #1E88E5)`
- **白色图标和文字**：在蓝色背景上清晰可见
- **活跃状态**：选中项使用海尔蓝渐变背景 + 阴影效果
- **用户项左侧边框**：3px 海尔蓝左边框标识选中状态
- **树形结构**：支持业务系统分组、用户列表展示

### 3. 消息气泡
#### 用户消息（右侧）
- **背景**：海尔蓝渐变 `linear-gradient(135deg, #006BB7, #1E88E5)`
- **白色文字**：清晰对比
- **圆角**：16px（右下角 4px）
- **阴影**：`0 2px 8px rgba(0, 0, 0, 0.08)`

#### AI助手消息（左侧）
- **背景**：白色
- **边框**：2px 海尔蓝浅色边框
- **图标**：白色背景 + 海尔蓝图标 + 边框
- **圆角**：16px（左下角 4px）

#### 工具消息
- **背景**：`#FFF8E1` 黄色系
- **边框**：`#FFE082`
- **图标**：橙色渐变

### 4. 头部工具栏
- **简化设计**：只保留左侧上下文信息和右侧"新建会话"按钮
- **移除**：模型选择器和审批模式（移至输入区域）
- **边框**：底部 2px 海尔蓝浅色边框
- **阴影**：轻微阴影提升层次

### 5. 输入区域（OpenUI 风格核心）
#### 设置栏
- **位置**：输入框上方
- **内容**：模型选择下拉框 + 分隔线 + 审批模式单选按钮组
- **样式**：
  - 渐变背景：`linear-gradient(135deg, #f8fbfd, #ffffff)`
  - 圆角：12px
  - 边框：1px 海尔蓝浅色
  - 单选按钮：海尔蓝选中态

#### 输入框
- **圆角**：24px（类似 OpenUI 的胶囊式设计）
- **边框**：2px 海尔蓝浅色
- **聚焦态**：
  - 边框变为海尔蓝主色
  - 外发光：`0 0 0 3px rgba(0, 107, 183, 0.1)`
- **自适应高度**：1-5 行

#### 发送按钮
- **圆形**：44px × 44px
- **背景**：海尔蓝渐变
- **阴影**：`0 4px 12px rgba(0, 107, 183, 0.3)`
- **悬停效果**：上移 2px + 加深阴影
- **点击效果**：回弹动画

### 6. 动画效果
- **消息进入**：`slideIn` 动画（淡入 + 上移）
- **打字动画**：3 个点的呼吸效果（海尔蓝）
- **欢迎图标**：`pulse` 脉冲动画
- **按钮悬停**：平滑过渡 + transform

### 7. 响应式设计
#### 桌面端（>1024px）
- 侧边栏：260px
- 消息最大宽度：900px
- 完整显示所有控件

#### 平板端（768-1024px）
- 侧边栏：220px
- 模型选择器：150px
- 单选按钮文字缩小

#### 移动端（<768px）
- 侧边栏：280px（抽屉式，默认隐藏）
- 遮罩层：半透明海尔蓝 + 模糊效果
- 设置栏：纵向排列
- 模型和审批模式：全宽显示
- 隐藏：提示文字、"新建会话"文字标签

### 8. 滚动条美化
- **宽度**：6px
- **滑块**：海尔蓝渐变
- **轨道**：海尔蓝浅色背景

### 9. 工具调用和审批卡片
#### 工具芯片
- **背景**：白色渐变
- **边框**：2px 海尔蓝浅色
- **阴影**：轻微阴影
- **悬停**：加深阴影

#### 审批卡片
- **背景**：黄色渐变 `linear-gradient(135deg, #FFF8E1, #FFFAED)`
- **边框**：2px `#FFE082`
- **图标**：橙色
- **按钮**：圆角按钮（批准-绿色 / 拒绝-红色）

## 与 OpenUI 对标的设计元素

1. ✅ **输入框在底部**，带设置选项（模型、参数）
2. ✅ **胶囊式输入框**（24px 圆角）
3. ✅ **简洁的头部**（最小化工具栏）
4. ✅ **大圆角消息气泡**（16px）
5. ✅ **渐变色使用**（海尔蓝替代原绿色）
6. ✅ **流畅的动画**（消息进入、悬停反馈）
7. ✅ **清晰的层次**（阴影和边框）
8. ✅ **响应式设计**（支持多端）

## 用户体验优化

- **视觉聚焦**：对话内容为中心，周边元素弱化
- **操作便捷**：常用设置在输入框附近，减少鼠标移动
- **状态清晰**：活跃项、悬停态、禁用态都有明确视觉反馈
- **加载反馈**：打字动画、工具执行状态、流式输出
- **触摸友好**：移动端按钮尺寸足够大（44px）

## 技术实现

- **Vue 3 Composition API**
- **Element Plus 组件库**
- **CSS 变量统一主题色**
- **Flexbox 布局**
- **CSS 动画和过渡**
- **媒体查询响应式**
- **SSE 流式对话**
